State Buddy.

Stop Wasting Time

Too much time of your time is spent clicking buttons, dragging sliders, and waiting for API calls to finish to get you application in the state you are currently developing for. With State Buddy, you can save all your states and recall them instantly.

Setup

In order to work with State Buddy, your app needs 2 things:

      A centralized store that holds onto all application state
      Methods to get and set that store

That's it! In your centralized store just include this snippet:

// Wire in State-Buddy window.addEventListener('message', function() { switch (event.data.type) { case 'GET_STATE': event.target.postMessage({ type: 'SEND_STATE', state: JSON.stringify(_getStore()) // Replace this with your method to get application state }, '*'); break; case 'SET_STATE': state = JSON.parse(event.data.state); _setStore(state); // Replace this with your method to set application state break; default: break; } });
